# Tmux split nav plugin

## Requirements

[Tmux plugin manager](https://github.com/tmux-plugins/tpm)

## Installation

```
set -g @tpm_plugins " \
  tmux-plugins/tpm \
  clauswitt/tmux_split_nav \
"
run-shell ~/.tmux/plugins/tpm/tpm
```
